◆ Card

typedef struct Card Card

Représente un mot dans la grille de Codenames.

Paramètres
wordtexte du mot (terminé par \0).
teaméquipe à laquelle le mot appartient (TEAM_*).
typetype de mot (masculin, féminin, etc.) pour l'affichage de la carte.
rectposition et taille de la carte à l'écran (calculé lors du rendu).
guess_rectposition et taille de l'icône de devinette à l'écran.
revealed0 si caché, 1 si révélé.
selected0 si non sélectionné, 1 si sélectionné.
is_pressed0 si non pressé, 1 si pressé.
is_hovered0 si non survolé, 1 si survolé.
display_word_once_revealedcôté client seulement, indique si le mot doit être affiché une fois la carte révélée (utile pour les cartes sélectionnées mais pas encore devinées)

◆ CardType

typedef enum CardType CardType

Types de cartes pour l'affichage (masculin, féminin, etc.). Ne correspond pas à une logique de jeu, mais uniquement à l'affichage. Permet d'avoir des cartes avec des formes différentes selon le type, pour une meilleure lisibilité.

Paramètres
CT_MALEcarte de type masculin.
CT_FEMALEcarte de type féminin.
CT_CATcarte de type chat.
CT_DOGcarte de type chien.
