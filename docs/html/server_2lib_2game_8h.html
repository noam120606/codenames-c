typedef enum CardType CardType

Types de cartes pour l'affichage (masculin, féminin, etc.). Ne correspond pas à une logique de jeu, mais uniquement à l'affichage. Permet d'avoir des cartes avec des formes différentes selon le type, pour une meilleure lisibilité.

Paramètres
CT_MALEcarte de type masculin.
CT_FEMALEcarte de type féminin.
CT_CATcarte de type chat.
CT_DOGcarte de type chien.

◆ Team

typedef enum Team Team

TEAM est utilisé à la fois pour catégoriser les mots dans la grille et pour assigner les joueurs à une équipe. Catégories de mots dans la grille de Codenames. Les mots sont classés en 4 catégories :

Paramètres
TEAM_NONEmot neutre (aucune équipe).
TEAM_REDmot appartenant à l'équipe rouge.
TEAM_BLUEmot appartenant à l'équipe bleue.
TEAM_BLACKmot assassin (met fin à la partie si révélé).
