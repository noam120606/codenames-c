Description détaillée

Configuration pour créer un Button. Tous les champs ont des valeurs par défaut définies via button_config_init. La structure peut être modifiée à tout moment et passée à button_create.

Champs configurables :

Paramètres
xOffset horizontal relatif au centre de la fenêtre (pixels). Positif → vers la droite, négatif → vers la gauche.
yOffset vertical relatif au centre de la fenêtre (pixels). Positif → vers le haut, négatif → vers le bas. Si x = 0 et y = 0, le centre du bouton est au centre de la fenêtre. Formules appliquées dans button_create : screen_x = WIN_WIDTH/2 + x - w/2 screen_y = WIN_HEIGHT/2 - y - h/2
wLargeur du bouton. Si 0 et text != NULL, calculée automatiquement selon le ratio de l'image de fond.
hHauteur du bouton (et hauteur cible pour le texte si is_text).
textTexte à afficher sur le bouton (UTF-8). NULL = bouton image sans texte.
font_pathChemin vers le fichier .ttf utilisé pour rendre le texte. Ignoré si text est NULL.
colorCouleur du texte. Ignorée si text est NULL.
tex_pathChemin vers l'image de fond (assets/img/…). NULL = pas de fond personnalisé (utilise "assets/img/buttons/empty.png" par défaut si text != NULL).
callbackFonction appelée lors du clic. Peut être NULL.
click_soundSon joué lors du clic (SoundID). Valeur négative = aucun son.
hover_soundSon joué lors du survol (SoundID). Valeur négative = aucun son.
hover_textTexte à afficher lors du survol (tooltip). NULL = aucun texte.

Champs d'état (runtime, gérés en interne — ne pas modifier directement) :

Paramètres
rectRectangle de rendu calculé depuis x/y/w/h.
textureTexture de fond chargée depuis tex_path.
is_hoveredMis à jour automatiquement lors des événements souris.
is_pressedMis à 1 quand un clic gauche commence sur le bouton.
is_textMis à 1 automatiquement si text != NULL.
text_rectRectangle de rendu du texte, calculé automatiquement.
text_textureTexture du texte, générée automatiquement depuis text/font_path/color.
rendererRenderer SDL utilisé pour charger les textures (doit être fourni à button_create).
text_dirtyIndique que le texte doit être régénéré (par exemple après une modification de la config). Mis à 1 automatiquement si text/font_path/color sont modifiés.
hover_text_textureTexture du texte de survol (tooltip), générée automatiquement depuis hover_text/font_path/color.
hover_text_rectRectangle de rendu du texte de survol, calculé automatiquement.
hover_start_ticksTimestamp du début du survol (SDL_GetTicks), utilisé pour afficher le tooltip après un délai.
hover_delay_msDélai en ms avant d'afficher le tooltip lors du survol.
