◆ MessageType

typedef enum MessageType MessageType

Types de messages harmonisé client/serveur (aussi appelé HEADER)

Paramètres
MSG_SERVER_ERRORmessage d'erreur envoyé par le serveur en cas de problème (ex: format de message invalide, action interdite, etc.). Le message d'erreur est suivi d'une description de l'erreur pour aider au debug.
MSG_UNKNOWNmessage inconnu.
MSG_CREATELOBBYmessage de création de lobby.
MSG_JOINLOBBYmessage de rejoindre un lobby.
MSG_LEAVELOBBYmessage de quitter un lobby.
MSG_LOBBYCLOSEDmessage de fermeture de lobby.
MSG_PLAYERJOINEDmessage d'un joueur qui a rejoint le lobby.
MSG_PLAYERLEFTmessage d'un joueur qui a quitté le lobby.
MSG_CHOOSE_ROLEmessage de choix de rôle/équipe dans le lobby (pour informer les autres joueurs en temps réel du choix d'un joueur).
MSG_STARTGAMEmessage de démarrage de partie.
MSG_WORDDATAmessage de données de mots pour la partie (envoyé par le serveur au démarrage de la partie, contient la liste des mots et leur couleur).
MSG_REQUESTUUIDmessage de demande d'UUID (pour le client de demander son UUID au serveur après connexion, si besoin pour le protocole).
