Description détaillée

Représente un tour de jeu pour une équipe.

Paramètres
spy_nameNom du joueur qui a soumis l'indice.
hint_countNombre de mots associés à l'indice (0 si indisponible).
agent_nameNom du joueur qui a révélé le(s) mot(s) du tour (vide si inconnu).
hintIndice soumis par l'espion (vide si indisponible).
revealed_wordsMots révélés pendant ce tour (tableau de taille NB_WORDS).
revealed_word_countNombre de mots révélés pendant ce tour.
revealed_word_teamsTEAM de chaque mot révélé (TEAM_NONE si inconnu).
